# Custom UI Components
- **Trust Shield**: Floating badge representing provider verification.
- **FairPrice Meter**: Segmented visual dial indicating pricing transparency (Excellent/Fair/High).
- **Map Preview**: Mock grid visual showing sector coordinates, travel paths, distance indicators.
- **Price Breakdown**: Accordion showing base, travel, complexity, and loyalty discount components.
- **Agent Trace Card**: Individual expandable list item showing agent inputs, decisions, rationale.
