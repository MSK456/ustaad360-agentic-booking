# Interactive Demo Flow
The app includes 6 built-in deterministic scenarios for judges to review:
1. **Scenario A (Successful Booking)**: Perfect end-to-end flow with high-scoring provider.
2. **Scenario B (Budget Mismatch Recovery)**: User requests service below the provider's quote. The system raises a red card and shows recovery alternatives (e.g. choose standard provider or expand budget).
3. **Scenario C (Roman Urdu Parsing)**: User queries using complex Urdu jargon. IntentAgent correctly identifies details.
4. **Scenario D (No Provider Available)**: Standard graceful failure trace.
5. **Scenario E (Provider Cancellation)**: Mid-trip cancellation and auto-recovery.
6. **Scenario F (Price Dispute)**: Launches AI-arbitrated Dispute Center.
