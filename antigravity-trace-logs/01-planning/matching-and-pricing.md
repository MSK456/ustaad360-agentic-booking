# Matching and Pricing Model

## 10-Factor Ranking Formula
```
Score = (Skill * 0.14) + (Reliability * 0.13) + (BayesianRating * 0.12) + (ReviewRecency * 0.11) + (Distance * 0.10) + (TravelTime * 0.09) + (PriceFit * 0.08) + (LoadCapacity * 0.07) + (LowCancellation * 0.05) + (EmergencyFit * 0.11)
```

## Dual Pricing Model
- **Services**: `Price = Base Rate + Travel Surcharge (Rs. 15/km) + Complexity Fee + Urgency Surcharge - Loyalty Discount`
- **Essentials (Groceries/Meat/Fruits)**: `Price = Sum(Quantity * Unit Price) + Delivery Surcharge (Rs. 200)`
